# Lista 4 - Estrutura de Dados I (Java) 📑

Repositório destinado à resolução da **Lista de Exercícios 4** da disciplina de Estrutura de Dados I, do curso de **Tecnologia em Sistemas para Internet (IFTM - Campus Patrocínio)**.

## 🚀 Sobre o Projeto

Esta lista foca na manipulação de **Arrays (Vetores)** e na implementação de algoritmos eficientes, com ênfase em estratégias **In-place** (manipulação direta na memória sem vetores auxiliares).

* **IDE:** IntelliJ IDEA
